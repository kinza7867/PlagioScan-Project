🚀 REST API Documentation

Integrate powerful plagiarism detection into your applications

POST Authentication

Get your API key by logging in or signing up. All API requests require authentication.

POST /api/login
Headers:
  Content-Type: application/json

Body:
{
  "email": "user@example.com",
  "password": "yourpassword"
}

Response:
{
  "success": true,
  "api_key": "your-api-key-here",
  "user": {
    "email": "user@example.com",
    "name": "John Doe",
    "plan": "pro"
  }
}

POST Check Plagiarism

Compare two texts and get comprehensive similarity analysis.

POST /api/check
Headers:
  Content-Type: application/json
  X-API-Key: your-api-key-here

Body:
{
  "text1": "Your original text here",
  "text2": "Text to check here"
}

Response:
{
  "success": true,
  "similarity_percentage": 45.2,
  "cosine_similarity": 42.5,
  "jaccard_similarity": 38.7,
  "uniqueness_score": 78.3,
  "verdict": "MODERATE SIMILARITY",
  "recommendation": "Review matching sections"
}

POST Upload File

Upload documents or images for text extraction and analysis.

POST /api/upload
Headers:
  X-API-Key: your-api-key-here

Body (multipart/form-data):
  file: your-file.pdf

Response:
{
  "success": true,
  "text": "Extracted text content",
  "word_count": 1250,
  "file_type": "pdf"
}

GET Check Status

💻 Code Examples

import requests

# Initialize API client
api_key = "your-api-key"
headers = {"X-API-Key": api_key, "Content-Type": "application/json"}

# Check plagiarism
data = {
    "text1": "Your original content",
    "text2": "Content to check"
}

response = requests.post(
    "https://api.plagioscan.com/api/check",
    headers=headers,
    json=data
)

result = response.json()
print(f"Similarity: {result['similarity_percentage']}%")
print(f"Verdict: {result['verdict']}")
// Using fetch API
const apiKey = 'your-api-key';
const response = await fetch('https://api.plagioscan.com/api/check', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': apiKey
  },
  body: JSON.stringify({
    text1: 'Your original content',
    text2: 'Content to check'
  })
});

const result = await response.json();
console.log(`Similarity: ${result.similarity_percentage}%`);
curl -X POST https://api.plagioscan.com/api/check \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{
    "text1": "Your original content",
    "text2": "Content to check"
  }'
